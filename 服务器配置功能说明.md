## 在 Xcode 中添加文件的步骤

1. 打开 Xcode 项目：`redredvideo.xcodeproj`
2. 在左侧项目导航器中，找到 `redredvideo/Network` 文件夹
3. 右键点击 `Network` 文件夹，选择 "Add Files to redredvideo..."
4. 在弹出的文件选择器中，导航到 `/Users/tian/Desktop/redredvideo/redredvideo/Network/`
5. 选中以下两个文件：
   - `RRServerConfigViewController.h`
   - `RRServerConfigViewController.m`
6. 确保勾选：
   - ✅ "Copy items if needed"（如果需要复制项目）
   - ✅ "Create groups"（创建组）
   - ✅ "Add to targets: redredvideo"（添加到目标）
7. 点击 "Add" 按钮

## 功能说明

### 首次启动
- App 首次启动时，如果没有配置过服务器 IP，会自动弹出配置页面
- 用户输入 IP 地址和端口（默认 3000）
- 点击确认后，配置会保存到 UserDefaults

### 后续使用
- 配置会自动加载，无需每次输入
- 如果需要修改服务器地址，可以调用：`[RRNetworkManager showServerConfig]`

### 配置存储
- IP 地址：`ServerIP` (UserDefaults)
- 端口：`ServerPort` (UserDefaults)
- baseURL 格式：`http://IP:PORT`

## 测试步骤
1. 删除 App 重新安装（清除 UserDefaults）
2. 启动 App，应该会弹出服务器配置页面
3. 输入服务器 IP（例如：192.168.1.100）
4. 点击确认
5. 检查网络请求是否使用了新的 IP

## 注意事项
- 确保手机和服务器在同一局域网内
- IP 格式验证：必须是 xxx.xxx.xxx.xxx 格式
- 端口默认为 3000，可以修改
