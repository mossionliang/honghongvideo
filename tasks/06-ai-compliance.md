# AI合规引擎（比赛核心模块）

> ⭐ 这是AI比赛的核心模块，所有AI能力集中在此。包括资质智能核验、内容风险识别、备案号闸门、审核流程引擎、AI增值功能五大部分。

---

## 一、AI服务基础

### AI-001 AI服务基础框架
- **优先级**: P0
- **预估工时**: 3天
- **依赖**: 无
- **描述**: 搭建AI服务的基础框架，封装多个AI模型的调用接口（OCR、图像识别、NLP、LLM），统一错误处理、重试、超时、并发控制。
- **技术要点**:
  - 统一的 AIService 抽象层，支持多 Provider（百度AI、阿里云AI、腾讯AI、OpenAI/Claude）
  - 异步任务队列（Celery/RQ），AI 审核任务入队异步处理
  - 结果缓存：相同素材不重复审核
  - 降级策略：主 Provider 不可用时自动切换备用
  - 调用计费追踪：记录每次AI调用的成本
- **涉及文件**: `backend/ai/base/`, `backend/ai/providers/`, `backend/ai/queue/`
- **验收标准**:
  - [ ] 多 Provider 可切换
  - [ ] 异步队列正常运行
  - [ ] 降级策略生效
  - [ ] 调用计费记录准确

## 二、资质材料智能核验

### AI-002 OCR识别-营业执照
- **优先级**: P0
- **预估工时**: 3天
- **依赖**: AI-001
- **描述**: 对合作方上传的营业执照图片进行OCR识别，自动提取关键字段并结构化。
- **技术要点**:

  - 置信度输出：每个字段的识别置信度，低于阈值标记为待人工确认
- **涉及文件**: `backend/ai/ocr/business_license.py`
- **验收标准**:
  - [ ] 识别准确率 > 95%
  - [ ] 所有关键字段提取完整
  - [ ] 低置信度字段正确标记

### AI-007 资质字段交叉校验
- **优先级**: P0
- **预估工时**: 3天
- **依赖**: AI-006
- **描述**: 对提取的资质信息进行交叉校验：营业执照公司名 vs 许可证持证单位 vs 入驻信息的一致性；版权授权链是否完整闭合。
- **技术要点**:
  - 名称一致性：模糊匹配（考虑括号、简称、全称差异）
  - 法人一致性：营业执照法人 vs 许可证法人
  - 授权链闭合：版权方 → 授权给 → 合作方，链路是否完整
  - 校验结果：通过/疑似不一致/严重不一致
  - 不一致项高亮展示给审核人员
- **涉及文件**: `backend/ai/verification/cross_check.py`
- **验收标准**:
  - [ ] 名称模糊匹配准确率 > 90%
  - [ ] 授权链闭合检查正确
  - [ ] 不一致项明确标记

  - 续期流程：合作方上传新证件后恢复
- **涉及文件**: `backend/ai/verification/expiry_monitor.py`
- **验收标准**:
  - [ ] 预警时间节点准确
  - [ ] 到期自动暂停生效
  - [ ] 续期后自动恢复

### AI-009 图像篡改/PS检测
- **优先级**: P2
- **预估工时**: 3天
- **依赖**: AI-001
- **描述**: 检测上传的资质图片是否存在PS痕迹或篡改嫌疑。
- **技术要点**:
  - ELA (Error Level Analysis)：检测JPEG压缩差异
  - 元数据分析：EXIF信息检查（编辑软件、修改时间）
  - 字体/排版异常检测：关键字段区域的字体一致性
  - 对接第三方图像鉴伪API（可选）
  - 输出：篡改概率分(0-100) + 疑似篡改区域标注
- **涉及文件**: `backend/ai/verification/tampering_detect.py`
- **验收标准**:
  - [ ] 明显PS痕迹检出率 > 80%
  - [ ] 误报率 < 10%
  - [ ] 疑似区域标注准确

### AI-010 主体黑名单匹配

### AI-018 风险分计算引擎
- **优先级**: P0
- **预估工时**: 3天
- **依赖**: AI-011~AI-017
- **描述**: 汇总所有AI审查结果，多维度加权计算综合风险分(0-100)。
- **技术要点**:
  - 维度权重（可配置）：涉黄×30 + 暴力×20 + 未成年×25 + 极端×25 + 文本×15 + 价值观×15
  - 单维度封顶：任一维度达到"确定违规"直接封顶100分
  - 取最高分 vs 加权平均（可配置策略）
  - 分数区间：0-20安全 / 20-60疑似 / 60-100高危
  - 输出：总分 + 各维度分数 + 决策建议
- **涉及文件**: `backend/ai/scoring/risk_calculator.py`
- **验收标准**:
  - [ ] 多维度加权计算正确
  - [ ] 封顶规则生效
  - [ ] 权重可配置
  - [ ] 分数区间划分合理

### AI-019 证据链生成

- **预估工时**: 2天

- **涉及文件**: `backend/ai/evidence/`
- **验收标准**:
  - [ ] 证据链完整关联
  - [ ] 截图清晰可辨
  - [ ] 证据包可下载
